KombiVojager
============

Kilka prostych implementacji rozwiązanie problemu komiwojażera.
Od najprostszych, po bardziej złożone.

1. Brutal
2. Naiwniak
3. Przekładaniec
    - najbliższy przodem
    - najdalszy przodem
4. Zwycięzca (dziel i zwyciężaj)
5. Mrówkojad
6. Genetyk
7. Internauta

Sposób użycia
=============

`python runner.py test_task_asymetric_distances_only`

gdzie `test_task_asymetric_distances_only` to nazwa przykładowego problemu
do rozwiązania.
